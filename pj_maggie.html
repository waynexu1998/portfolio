Maggie is a marketing and design specialist who graduated from University of Rochester in 2020. This project was a prototype built for her with an intended purpose of being her portfolio website. In the process of designing the website we collaborated to design a magazine-like layout and smooth page transitions. Adding a constant grain was a fine tuned touch to bring out more texture for the website. Implementing the page transitions using framer API also made it a much more pleasant expereience to browse through. Using Next.js as the chosen framework, this project became a stunning prototype.
